Introducción:

1) Podriamos utilizarlo al igual que se podria usar Block de Notas (notepad.exe) o Notepad++ como editores de TEXTO, pero
la realidad es que es útil o necesario utilizar un editor de código como lo es Visual Studio Code donde te recomienda
ciertas etiquetas, te ayuda con el auto-completado y ademas indica ciertas veces cuando estamos incertando mal una etiqueta ya sea escribiendo mal su sintaxis o bien mencionandola. Por otro lado un editor de código posee las herramientas
para procesar el código o bien ejecutarlo.

2) Las diferencias entre Front-End, Back-End y FullStack.
El desarrollador Front-End por lo general trabaja sobre la planilla HTML, la estilizacion de la página y un poco con código y algoritmos de Javascript o algún otro lenguaje donde estas herramientas se logren visualizar en pantalla a modo
estático, existen algunas herramientas con el manejo del DOM para cambiar un texto o un objeto frente a nuestro index.html
pero al recargar nuevamente la misma, todo vuelve a su estado inicial.
El desarrollador Back-End se encarga de trabajar en todo lo que es lado de Servidor, lo que el usuario no ve, palabras en ingles, código extraño, funciones complejas, etc, signos o caracteres como parentesis, corchetes y llaves para las funciones, métodos,etc. Ademas de realizar conexiones a base o bases de datos, realizar consultas, insertar informacion, editar o eliminar en una base de datos y/o tablas. Como tambien obtener informacion de otros servidores/páginas.
El FullStack seria la combinacion o fusión entre el Front-End y el Back-End.
